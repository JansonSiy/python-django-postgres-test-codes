HOW TO INSTALL DJANGO?
Video Link: https://www.youtube.com/watch?v=e_o1nacGQiw

STEP 1: Download Brew first
Link: https://brew.sh/
To check brew version: brew -v

STEP 2: Install Python
Code: brew install python3
To check python 3 version: python3

STEP 3: Python 3 comes with pip 
To check pip version: pip3
Source: https://stackoverflow.com/questions/40868345/checking-whether-pip-is-installed

STEP 4.a: Create a folder then enter that folder
STEP 4.b: Setup Virtual Environment (per project)
To create a virtual environment: python3 -m venv venv_blogApp (python3 -m venv NAME)

STEP 5: Activate Virtual Environment(go to parent dir)
To activate virtual environment: source venv_blogApp/bin/activate (source NAME/bin/activate)
To deactivate: deactivate

STEP 6: Install Django using pip
Code: pip3 install django

STEP 7: Running the server
Link: https://stackoverflow.com/questions/25664654/issues-in-initial-setup-for-django-project

Create a project file first: django-admin startproject projectFolder_blogApp
Enter the project folder->
To access the server: python manage.py runserver 8000

STEP 8: creating an app
Go inside the projectFolder with the manage.py file and paste this in the terminal. This will create an app
code: python manage.py startapp name

HOW TO INSTALL REDIS?
Link: https://redis.io/
Video Link: https://www.youtube.com/watch?v=lgWjGkdrExA
Manual Installation: https://www.youtube.com/watch?v=vPcFJvvJgVA

STEP 1: Update brew first before anything else
code: brew update

STEP 2: Use brew to install redis
code: brew install redis

STEP 3: Start the service
code: brew services start redis
code: redis-cli ping (double check if it's really running, should return pong)

To stop the service: brew services stop redis
To uninstall redis: brew uninstall redis

HOW TO USE SSH?
STEP 1: Generate SSH Key
code (in terminal): ssh-keygen -t rsa

STEP 2: Copy code
code: pbcopy < ~/.ssh/id_rsa.pub

STEP 3: Go to settings-SSH and GPG keys-New SSH key-Add SSH key(paste in the key field)

STEP 4: git clone ssh-link
