Python Data Structures: Dictionary Object

How to create a dictionary?
Example: d = {}

A key can be a string or an integer. If you enter a specific key it will return the value.
Example: d[1] = "yes" or d['1'] = "no" or d[2] = 9000

Reiterating through the dictionary:
Example:
d.keys() = ['1', 1, 2] (returns all the keys in the list)
d.items() = [('1', 'no'), (1, 'yes'), (2, 9000)] (returns all the key-value pair)
print d = {'1': 'no', 1: 'yes', 2: 9000} (simply print)

Python Data Structures: Python List Methods

Example: fruits = ['apple', 'grapes', 'mango']

#1 .insert() - inserting an element on the array (referencing on the index declared)
code: fruits.insert(2, 'banana')
output: fruits = ['apple', 'grapes', 'banana', 'mango']

#2 .append() - append an element at the end of an array
code: fruits.append('banana')
output: fruits = ['apple', 'grapes', 'mango', 'banana']

#3 .pop() - display the last element of an array and removes it
code: fruits.pop()
output: 'mango'
output2: fruits = ['apple', 'grapes']
note: please note that you can also indicate an index parameter (ex. fruits.pop([0]) = ['grapes', 'mango'])

#4 .reverse() - reverses the elements inside the array
code: fruits.reverse()
output: ['mango', 'grapes', 'apple']

#5 .count() - counts the number of the mentioned element in the array
example: fruits = ['apple', 'apple', 'mango']
code: fruits.count('apple')
output: 2

#6 .copy() - literally copies an array
code: fruits2 = fruits.copy()
output: fruits2 = ['apple', 'grapes', 'mango']

#7 .extend() - combine the elements of the 2 arrays
example: fruits2 = ['orange', 'watermelon', 'coconut']
code: fruits.extend(fruits2) or fruits + fruits2
output: fruits = ['apple', 'grapes', 'mango', 'orange', 'watermelon', 'coconut']

#8 .sort() - will sort out the elements in the array (please not that in order for this to work, datatypes of the elements inside an array must be the same)
example: fruits2 = ['orange', 'watermelon', 'coconut']
code: fruits2.sort()
output: ['coconut', 'orange', 'watermelon']

#9 .remove() - removes the declared element
code: fruits.remove('apple')
output: ['grapes', 'mango']

#10 .clear() - delete all elements from an array
code: fruits.clear()
output: fruits = [];

#11 .index()
example: janson = ['siy', 'Pasig', 'siy', 27]

code: janson.index('siy')
output: 0
notes: will stop upon first occurence

code: janson.index('siy', 1)
output: 2
notes: ignored the 'siy' = index 0 because you told it to start at index 1

example: jeff = [1, 2, 1, 1, 3, 4, 1]
code: jeff.index(1, 2)
output: 2
notes: stopped at the first occurence

code: jeff.index(1, 4, 7)
output: 6
notes: started to look for 1 from index 4 to 7 then returned the index number of the first occurence
