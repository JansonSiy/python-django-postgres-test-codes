Python Data Structures: Python List Methods

Example: fruits = ['apple', 'grapes', 'mango']

 #1 .insert() - inserting an element on the array (referencing on the index declared)
 code: fruits.insert(2, 'banana')
output: fruits = ['apple', 'grapes', 'banana', 'mango']

#2 .append() - append an element at the end of an array
 code: fruits.append('banana')
output: fruits = ['apple', 'grapes', 'mango', 'banana']

#3 .pop() - display the last element of an array and removes it
 code: fruits.pop()
output: 'mango'
output2: fruits = ['apple', 'grapes']
note: please note that you can also indicate an index as a parameter (ex. fruits.pop([0]) = ['grapes', 'mango'])

#4 .reverse() - reverses the elements inside the array
 code: fruits.reverse()
output: ['mango', 'grapes', 'apple']

#5 .count() - counts the number of the mentioned element in the array
 example: fruits = ['apple', 'apple', 'mango']
code: fruits.count('apple')
output: 2

#6 .copy() - literally copies an array
 code: fruits2 = fruits.copy()
output: fruits2 = ['apple', 'grapes', 'mango']

#7 .extend() - combine the elements of the 2 arrays
 example: fruits2 = ['orange', 'watermelon', 'coconut']
code: fruits.extend(fruits2) or fruits + fruits2
output: fruits = ['apple', 'grapes', 'mango', 'orange', 'watermelon', 'coconut']

#8 .sort() - will sort out the elements in the array (please not that in order for this to work, key datatypes of the elements inside an array must be the same)
 example: fruits2 = ['orange', 'watermelon', 'coconut']
code: fruits2.sort()
output: ['coconut', 'orange', 'watermelon']

#9 .remove() - removes the declared element
 code: fruits.remove('apple')
output: ['grapes', 'mango']

#10 .clear() - delete all elements from an array
 code: fruits.clear()
output: fruits = [];

#11 .index()
 example: janson = ['siy', 'Pasig', 'siy', 27]

code: janson.index('siy')
output: 0

Using Lists as Stacks

The list methods make it very easy to use a list as a stack, where the last element added is the first element retrieved (“last-in, first-out”).
methods: .append() & .pop()

Using Lists as Queues

It is also possible to use a list as a queue, where the first element added is the first element retrieved (“first-in, first-out”); however, lists are not efficient for this purpose. While appends and pops from the end of list are fast, doing inserts or pops from the beginning of a list is slow (because all of the other elements have to be shifted by one).

methods:
append() :- This function is used to insert the value in its argument to the right end of deque.
appendleft() :- This function is used to insert the value in its argument to the left end of deque.
pop() :- This function is used to delete an argument from the right end of deque.
popleft() :- This function is used to delete an argument from the left end of deque.

example:
from collections import deque
de = collections.deque([1,2,3])

de.append(4)
The deque after appending at right is :
deque([1, 2, 3, 4])

de.appendleft(6)
The deque after appending at left is :
deque([6, 1, 2, 3, 4])

de.pop()
The deque after deleting from right is :
deque([6, 1, 2, 3])

de.popleft()
The deque after deleting from left is :
deque([1, 2, 3])

List Comprehensions

List comprehensions provide a concise way to create lists.

#1
code without List Comprehension:

for x in range(10): OR for x in range(1, 10):
squares.append(x**2)

output: squares = [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

or

shorthand code with List Comprehension:
squares = [x**2 for x in range(1, 10)]

#2
names = ["Janson", "Harold", "Jefferson", "Jetson", "Smith", "Kenrick"]

code without List Comprehension:

jnames.append(eachname)
